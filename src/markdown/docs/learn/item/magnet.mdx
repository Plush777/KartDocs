#### 관련 용어

- 짜부: 자석 + 부스터 조합의 줄임말.
- 빨렸어: 상대방이 자신에게 자석을 사용했을 때 사용하는 용어. 보통 이러한 브리핑이 들어오면 마지막 바퀴가 끝나기 전이기 때문에 <br/>
뒤에 팀원이 있다면 R키를 눌러 자석을 사용한 상대방이 가속을 받지 못하게 할 수도 있음. (드리프트에선 전자파로 방어하는 전략으로도 사용가능.)
- 알: 상대한테 자석이 빨렸으니 R키를 친다거나 쳐달라는 의미.
