<Bq>트랙 위에 바나나를 던집니다. <br/>
바나나를 밟으면 미끄러지며, 잠시 조작이 불가능해집니다.</Bq>

1등에서만 나오는 아이템으로, 바나나를 밟은 플레이어는 일정 시간동안 방향 조정이 불가능해집니다. <br/>
이러한 특성을 이용하여 주로 코너나 피하기 힘든 좁은 길에 설치합니다.

<br/>

팀전에서는 팀원에게도 피해를 줄 수 있으니 팀원이 뒤에 있다면, 밟기 힘든 곳에 설치하기도 합니다.
