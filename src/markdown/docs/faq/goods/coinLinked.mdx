---
title: 다른 플랫폼에서 충전한 K-COIN은 사용할 수 없나요?
---

image

코인은 현재 접속한 플랫폼에서 사용 가능한 코인만 표시되며,
**다른 플랫폼에서 구매한 코인과 연동되지 않습니다.**

<br/>

예를 들어, PC 플랫폼에서 코인을 충전했을경우 <br/>
안드로이드나 ios, 등의 플랫폼에서는 확인 및 사용이 불가능합니다.

<br/>

코인뿐만 아닌 다른 재화 (루찌, 마일리지) 들도 동일하게 적용됩니다.
