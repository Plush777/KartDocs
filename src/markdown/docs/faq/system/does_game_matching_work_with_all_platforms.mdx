---
title: 게임 매칭은 모든 플랫폼이랑 같이 되나요?
---

매칭은 플랫폼 (PC, 모바일, 콘솔) 에 관계없이 모두 같이 매칭됩니다.

콘솔 (Play Station, Xbox) 의 경우, 메뉴 - 환경설정 - 일반에서 크로스 플레이 설정을 할 수 있습니다.

- 크로스 플레이 사용 : 모든 플랫폼과 매칭
- 크로스 플레이 사용안함 : 같은 플랫폼끼리만 매칭
